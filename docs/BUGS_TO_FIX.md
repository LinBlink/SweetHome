# FRONT END BUGS

## 家庭动态页面
- 点赞页面不能显示“……人觉得很赞”。我们的设计是一个人可以点赞很多次。应该是显示“……次被点赞”。在点赞列表中，对于每个人要显示“某某某 点赞……次”
- 动态内容似乎没有做缓存？导致每一次媒体文件都要重新加载
- 家庭动态页面视频播放显示的静音按钮，点击之后可以播放声音
- 视频缩略图圆角没有做好。如果视频高度比较高，只有左侧有圆角，右侧没有


- 对聊天记录进行搜索时点击搜索对象无法直接定位到那一则聊天记录
- 设置页面增加删除本地聊天记录的功能（非服务器上的聊天记录）
- 主题切换没有应用到全局。切换主题时只有房子图标的颜色改了。
