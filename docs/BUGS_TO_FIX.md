# FRONT END BUGS

## 家庭动态页面

- 发动态可以保存草稿
- 聊天页面如果对图片进行放大，发现只能在图片尺寸下放大，无法放大到全屏
- 发动态时加上可以长按拖动改变顺序的提示
- 发动态时视频进行预览体验不佳，没有和动态页面中打开视频一样的播放控制
- 发动态时如果对图片进行放大，发现只能在图片尺寸下放大，无法放大到全屏


- 导出聊天记录时可以选择导出哪些对话的记录
- 导出聊天记录时可以选择通过第三方应用打开分享
- 导出聊天记录可以选择格式，txt（不带图片）和 pdf（带图片）
- 聊天页面可以选择多个图片发出（每个图片发送当作独立消息）
- 聊天页面可以选择发送视频、音频，视频可以调用用户相机或者flutter第三方控件进行录制。（视频 音频发送消息时消息type为video 和 voice，content为各自上传后的链接）
